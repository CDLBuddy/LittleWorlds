# World Layouts

This directory contains TypeScript modules that define object placements for game worlds. These layouts are generated by the WorldEditor tool (press F2 in-game while in DEV mode).

## Structure

Each layout file exports marker arrays grouped by category:
- `treeMarkers` - Tree/vegetation positions
- `interactableMarkers` - NPCs, clickable objects
- `propMarkers` - Decorations, static props
- `generalMarkers` - Uncategorized markers

## Usage

Import the layout in your world file:

```typescript
import { backyardLayout } from './layouts/backyard';

// Use markers to place objects
backyardLayout.tree.forEach(marker => {
  const tree = createTree(marker.position);
  tree.rotation.y = marker.rotation;
});
```

## Generating Layouts

1. Press **F2** to enable WorldEditor
2. Press **1-4** to switch categories
3. **Ctrl+Click** to place markers
4. **WASD/QE** to position/rotate markers
5. Press **Ctrl+E** to export layout module
6. Paste the output into a new file in this directory

## Marker Interface

```typescript
export interface LayoutMarker {
  position: Vector3;
  rotation: number; // Y-axis rotation in radians
  category: 'tree' | 'interactable' | 'prop' | 'general';
  label: string;
}
```

## Best Practices

- Keep layouts organized by area/world
- Use descriptive labels for markers
- Commit layouts to version control
- Reference layouts as the source of truth for placements
