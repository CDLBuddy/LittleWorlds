- **Data:** JSON star catalog (accurate RA/Dec, magnitude)
- **Shader:** Subtle twinkle (noise-based flicker, not disco)
- **Performance:** <500 tris (mostly billboards)

### Lantern Light
- **Source:** Dynamic point light (range 8, intensity 1.5)
- **Attached:** Follows player smoothly (no jitter)
- **Toggle:** Instant on/off (no fade, responsive)
- **Moths:** Optional particle effect (attracted to light, gentle)

### Sky Dome
- **Mesh:** UV sphere (inverted normals, player inside)
- **Texture:** Procedural star field + milky way overlay
- **Rotation:** Slow (accurate sidereal rate, optional)
